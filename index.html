01

Three steps 三步开始

Install once. Arrange in a second. 安装一次，一秒排好。

  1. 1

    Download the beta 下载 Beta

    Get version 1.0.0-beta.3 from the official GitHub pre-release. 从官方 GitHub 预发布页面获取 1.0.0-beta.3。

  2. 2

    Move it to Applications 移入“应用程序”

    Copy Auto Window Arranger to /Applications, then open it. 将 Auto Window Arranger 复制到 /Applications，然后打开。

  3. 3

    Grant Accessibility 授予辅助功能权限

    Choose the apps to arrange, then approve access when the app asks. 选择要排列的 App，并在应用说明用途后主动授权。

Accessibility辅助功能

macOS Accessibility permission lets Auto Window Arranger read window geometry and move the windows you selected. It does not read window titles, contents, keyboard input, or files. macOS 辅助功能权限让 Auto Window Arranger 读取窗口几何信息并移动你所选 App 的窗口； 应用不会读取窗口标题、正文、键盘输入或文件。

02

Exact geometry 精确几何

A layout for every window count. 每种窗口数量，都有清晰布局。

1Centered原尺寸居中
2Equal columns左右等分
3Three columns横向三等分
42 × 2 grid2 × 2 宫格
5Grid + center宫格 + 居中叠放
6+Balanced grid自动等分网格

Layouts use the usable area of the display under your pointer, keeping clear of the menu bar and Dock. Arrange one selected app at a time, cycle through running apps, or combine all selected apps in one layout. 布局使用鼠标所在显示器的可用区域，不遮挡菜单栏和 Dock。你可以一次排列一个所选 App、 在运行中的 App 之间循环，或把所有所选 App 合并到同一布局。

03

Privacy and support 隐私与支持

Your windows stay on your Mac. 你的窗口信息留在 Mac 上。

No background data transfer 不在后台传输数据

Window and app metadata is processed locally. There is no account, advertising, telemetry, automatic crash upload, cloud sync, or automatic update check. 窗口与 App 元数据只在本机处理。应用不含账号、广告、遥测、自动崩溃上传、云同步或自动更新检查。

Read the privacy policy 阅读隐私政策 ↗

Support with a clear boundary 边界清晰的支持渠道

Use GitHub Issues for non-sensitive problems. Send private support, privacy, or security details through the documented private channels. 非敏感问题请使用 GitHub Issues；私密支持、隐私或安全信息请通过文档中的私下渠道提交。
