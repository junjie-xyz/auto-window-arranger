# Auto Window Arranger

Official public distribution repository for Auto Window Arranger, a macOS menu bar app for arranging windows of user-selected apps.

Auto Window Arranger 的官方公开分发仓库。这是一款用于排列用户所选 App 窗口的 macOS 菜单栏应用。

## Downloads / 下载

No public version is available yet. Signed and notarized releases will be published on the [Releases](https://github.com/junjie-xyz/auto-window-arranger/releases) page after validation.

目前尚未发布公开版本。通过验收后，已签名并经过 Apple 公证的版本会发布到 [Releases](https://github.com/junjie-xyz/auto-window-arranger/releases) 页面。

This repository contains public distribution assets and documentation only. The application source code and private build records are not published here.

本仓库只包含公开分发资产和文档，不公开应用源码或私有构建记录。

## Support / 支持

- Report non-sensitive problems through [Issues](https://github.com/junjie-xyz/auto-window-arranger/issues).
- Do not post credentials, private logs, window contents, or other sensitive information in a public issue.
- 非敏感问题请通过 [Issues](https://github.com/junjie-xyz/auto-window-arranger/issues) 提交。
- 不要在公开 Issue 中提交凭据、私密日志、窗口内容或其他敏感信息。

## Security / 安全

Read [SECURITY.md](SECURITY.md) before reporting a vulnerability. Use GitHub's private vulnerability reporting form or email `ioriens@outlook.com`; do not create a public security issue.

报告漏洞前请阅读 [SECURITY.md](SECURITY.md)。请使用 GitHub 私密漏洞报告表单或发送邮件到 `ioriens@outlook.com`，不要创建公开安全 Issue。
