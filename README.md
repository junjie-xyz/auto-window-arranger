# Auto Window Arranger

Auto Window Arranger is a macOS menu bar app that precisely arranges every window from the apps you choose. Trigger a layout manually, use configurable global shortcuts, or let the app update the layout when windows open and close.

[Official website](https://junjie.xyz/auto-window-arranger/) · [中文](https://junjie.xyz/auto-window-arranger/?lang=zh)

## Download

Download the public beta from the dedicated [v1.0.0-beta.3 pre-release](https://github.com/junjie-xyz/auto-window-arranger/releases/tag/v1.0.0-beta.3). v1 is currently free.

### Requirements

- macOS 14 or later
- Apple Silicon is supported
- The Universal build includes Intel code, but Intel hardware has not yet been verified
- macOS Accessibility permission is required to inspect and move selected windows

## Window layouts

- 1 window: keep its size and center it
- 2 windows: equal columns
- 3 windows: three equal columns
- 4 windows: 2 × 2 grid
- 5 windows: four in a grid, with the fifth centered at the same size
- 6 or more windows: balanced grid

Layouts use the usable area of the display under the pointer and keep clear of the menu bar and Dock. The app does not preselect or require any third-party app.

## Privacy

Window and app metadata is processed locally. Auto Window Arranger has no account, advertising, telemetry, automatic crash upload, cloud sync, automatic update check, or automatic network request.

Read the complete [Privacy Policy](https://junjie.xyz/auto-window-arranger/privacy.html).
